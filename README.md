## About

This is a simple encryption tool written in C.

## Getting Started

Built with GCC. 

### Prerequisites

Be sure you have gcc installed

```sh
sudo apt-get install gcc
```

### Building

1. Compile using GCC
```sh
gcc main.c -o hexcrypt
```
2. Run
```sh
./hexcrypt
```
